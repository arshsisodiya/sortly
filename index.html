Built in the open.
Trusted by design.

Sortly is fully open source under the MIT license. Every line of code is public, auditable, and forkable. No hidden network calls. No telemetry. No surprises.

Your files never leave your machine. Sortly works entirely locally — it moves files between folders on your own drive and does nothing else.

🔒

100% offline

No internet connection required. No data ever sent anywhere. What's on your drive stays on your drive.

📖

Fully auditable

Read every line on GitHub. Fork it, modify it, compile it yourself. MIT license — do whatever you want.

↩

Non-destructive by default

Files are moved, never deleted. Undo is always one click away. Sortly never overwrites — conflicts are renamed safely.

# Sortly — open source, MIT license
# github.com/arshsisodiya

def categorize(file_path: str) -> str:
  """Map file → category by extension."""
  ext = Path(file_path).suffix.lower()

  for rule in self.custom_rules:
    if rule["pattern"] in name:
      return rule["category"]

  return self._ext_map.get(ext, "Others")

def execute_plan(plan, folder):
  # Move files — never delete
  for src, dst, cat in plan.moves:
    shutil.move(src, dst) # safe move
    history.push(src, dst) # always undoable
